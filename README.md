# Koban-Like-Demo

A simple Koban-like board application built with React and Firebase.

---

## Description

Koban-Like-Demo is a Kanban-style board application where users can create boards, add lists, and manage tasks. It uses Firebase for authentication and real-time database updates.

---

## Features

- User authentication with Firebase
- Create, edit, and delete boards, lists, and cards
- Drag-and-drop tasks between lists (on Desktop)
- Mark tasks as completed
- Responsive UI

---

## Tech Stack

- **Frontend:** React (with Hooks), React-Router, Tanstack Query, DND Kit
- **Backend:** Firebase (Auth, Firestore)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS

---

## Deployment

This project is deployed on Vercel: https://koban-like-demo.vercel.app
